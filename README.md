# DCU Python – CSC1034 & CSC1035

Python assignments and labs completed during Year 2 of BSc Computing for Business 
at Dublin City University (2025–2026).

## Topics Covered

- Core Python: strings, lists, tuples, dictionaries, sets
- Loops, range, and iteration
- Functions and built-in modules
- File I/O, exception handling, and datetime
- Object-oriented programming: classes, special methods, 
  operator overloading, and inheritance
- Recursion and sorting algorithms
- List comprehensions
- Data analysis with Pandas and Seaborn
- Graph theory: depth-first and breadth-first search
- Regular expressions
- Command line arguments and docstrings

## Structure

Each folder corresponds to a topic area and contains lab exercises 
and assignments from CSC1034 and CSC1035.

## Languages & Tools
- Python 3
- Pandas
